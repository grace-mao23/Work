Purple Pancakes Presents the Average Person's Guide to Foundation

Foundation is a basic front-end framework tool to get anyone started with their own webpage. Throughout this page, we will be demo-ing the different resources and tricks that come with this awesome front-end framwork.

In order to get started with foundation, please enter this code into your html page

<!-- Compressed CSS -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/foundation-sites@6.5.3/dist/css/foundation.min.css">

<!-- Compressed JavaScript -->
<script src="https://cdn.jsdelivr.net/npm/foundation-sites@6.5.3/dist/js/foundation.min.js"></script>

The key features we'd like to present you with are listed on the menu at the top!

That being said, let's look at how to create a menu first.

Menus: What Would You Like to Order?

Menus can help you organize your webpage and make it easy to navigate. Foundation makes it super easy for you to have a menu, because it's built in!

By using the class "menu", you can create a menu of your own choosing

Here's how the above menu was made →

<ul class="menu">

<li class="is-active"><a href="#">Home</a></li>
<li><a href="#intro">Introduction</a></li>
<!-- ... -->

</ul>

Now you might be wondering why we're showing you these random grids.
In fact, this whole webpage is a grid, specifically made up of different sized cells
These cells are invisible, but Foundation makes it easy to create them in order to format your website into different divisions. Using the "callout" tag creates the border. This way, you can easily separate sections and divide up the page.

Code snippet ↙

<div class="large-12 cell">
<div class="callout">
Here is a 12 cell section
</div>
</div>
